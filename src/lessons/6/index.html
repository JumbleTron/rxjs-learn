Operatory transformacji i filtrowania w RxJS

W tej lekcji poznajemy:

  • Operatory transformacji strumieni:
    • map() - przekształcanie wartości
    • filter() - filtrowanie wartości
    • tap() - podgląd wartości bez ich modyfikacji
    • debounceTime() - kontrola częstotliwości emisji
  • Obsługę błędów:
    • catchError() - obsługa i transformacja błędów
    • EMPTY - zakończenie strumienia bez emisji
  • Integrację z API zewnętrznym:
    • Użycie ajax do pobierania danych
    • Transformacja odpowiedzi API

Przykłady zawierają:

  • Symulację kanału wiadomości z filtrowaniem kategorii
  • Pobieranie losowych imion z API
  • Transformację i filtrowanie liczb
  • Kontrolę wprowadzania tekstu z debounce
  • Obsługę błędów z wartościami fallback

Interaktywna demonstracja:

  • Pole tekstowe z debounce - pokazuje kontrolę częstotliwości emisji
  • Konsola - demonstracja różnych operatorów i ich efektów
  • Przykład integracji z zewnętrznym API
